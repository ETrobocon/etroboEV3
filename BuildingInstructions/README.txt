README

ET ロボコンで使用する EV3way-ET の「非公式」な組み立て図です。
実際の大会で用いられる走行体とは組み立て方が異なりますのでご注意ください。

  EV3wayET16.pdf
  デベロッパー部門プライマリークラスで使用する、二輪倒立振子型の走行体。
  (2017年大会までのタイヤを用いたもの）
  
  EV3wayET18b.pdf
  デベロッパー部門プライマリークラスで使用する、二輪倒立振子型の走行体。
  (2018年大会以降のタイヤを用いたもの）

使用許諾（ライセンス）
以下の用途に限ってご利用下さい。

・ETロボコンへの参加のため
・ETロボコンへの参加を検討するため
・ETロボコンの広報
・ETロボコンに参加、協力していることを広報するため
(ご不明な点は、ETロボコン 本部事務局にお問合せください)

本組み立て図の著作権は、ETロボコン実行委員会に帰属します。
