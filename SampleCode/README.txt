README

ET ロボコンで使用する二種類の走行体（EV3way と ETrikeV）の、各プラットフォーム用のサンプルコードです。

  EV3way
  - EV3way_EV3RT_sample.tgz
    TOPPERS/EV3RT 用。 (C)

  - EV3way_leJOS_sample.zip
    leJOS 用。 (Java)

  - EV3way_MonoBrick_sample.zip
    MonoBrick EV3 Firmware 用。 (C#)

  ETrikeV
  - ETrikeV_EV3RT_sample.zip
    TOPPERS/EV3RT 用。 (C)

  - ETrikeV_leJOS_sample.zip
    leJOS 用。 (Java)

  - ETrikeV_MonoBrick_sample.zip
    MonoBrick EV3 Firmware 用。 (C#)


使用許諾（ライセンス）
BSD ライク、ないし、各プラットフォームのライセンスに従います。
